📚 ¿Cómo funciona el Algoritmo de Cristian?

  1. El cliente solicita la hora al servidor.
  2. El servidor responde con su hora actual.
  3. El cliente calcula el tiempo de ida y vuelta (RTT).
  4. El cliente estima el retardo de la red.
  5. El cliente ajusta su reloj utilizando la hora recibida.
